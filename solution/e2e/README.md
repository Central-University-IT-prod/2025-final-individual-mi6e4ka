# e2e тест

перед запуском желательно отчистить БД

### запуск:

```bash
go run ./cmd/app
```

### что проверяет

тест проверяет базовый путь рекламодателя и клиента:

- создание пользователей
- корректное/некорректное создание кампаний
- добавление ml-score
- получение объявления
- клик по объявлению
